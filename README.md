# 進め方・準備編
１．Ubuntu マシンにVagrantとVirtualboxを導入  
２．Vagrant 用のディレクトリを作って初期化  
３．何も設定せず試す

６．hostname と ip を設定

８．別ディレクトリで３つめのVMを立ててssh で接続  
９．Vagrant ファイルのまとめ
